Project was created with Android studio as part of a group project. Titled "Student Connect", this project is a fully operational messaging application designed to connect students in a University setting. Users are able to create accounts, make private chat rooms, create events, send and receive files/pictures in the chat rooms. Users are added into course chatrooms based on their selected enrolled courses, as well as registered events/hobbies.

Group operated with Agile - Scrum methodology and developed the project using Android Studio. 

Firebase API services was implemented into this application to allow for realtime database updates.
